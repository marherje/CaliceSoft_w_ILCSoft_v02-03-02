Detailed Description

This tool collects results from individual calibration measurements + temperatures for each channel from the specified range of runs and creates TGraphErrors from these values. Histograms of the operation voltages are included for xchecks.

Usage:

extractGainVsTemp <dbfolder> <tempCalib> <first> <last> <outfile> [<exclude>]
Parameters
[in]<dbfolder>DB folder containing results from the single gain measurements (e.g. "/cd_calice/Hcal/Calib_Measurements/gain_single")
[in]<tempCalib>DB folder containing calibration for temperature sensors (e.g. "/cd_calice/Hcal/tempSensors")
[in]<first>first run of run range
[in]<last>last run of run range
[out]<outfile>name of the output file (*.root)
[in]<exclude>(optional) .txt file with list of runs that are excluded from run range
