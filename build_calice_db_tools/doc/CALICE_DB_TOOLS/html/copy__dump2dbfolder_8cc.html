Detailed Description

This tool copies the whole content of a DB folder dump file to a database folder (all layers, tags and times).

Usage:

copy_dump2dbfolder <filename> <foldername>
Parameters
[in]<filename>input dump file (.slcio)
[out]<foldername>name of the DB folder where the content of the input file is stored
