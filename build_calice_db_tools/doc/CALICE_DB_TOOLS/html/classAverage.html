Detailed Description

template<class T>
class Average< T >

Collect input values and calculate mean/RMS upon request

This class keeps sum, sum-of-squares and number of input values to calculate mean/RMS. Templated value data type allows for flexible memory managemant, while internal calculations are done in double precision to avoid numerical cut-offs.
