Detailed Description

This tool copies the content of a DB folder dump or a DB file to a database folder (all layers, tags and times) and adds new parameters to all collections copied.

Usage:

copy_dump2dbfolder <filename> <foldername> <parameterfile_float>
Parameters
[in]<filename>input dump file (.slcio)
[out]<foldername>name of the DB folder where the content of the input file is stored
[in]<parameterfile_float>input file (.txt) containing the float parameters to append to all collections (format: name value)
