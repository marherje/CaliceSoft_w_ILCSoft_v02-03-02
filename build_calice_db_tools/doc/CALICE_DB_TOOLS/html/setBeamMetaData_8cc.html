Detailed Description

This tool stores the BeamMetaData information for one run in the database.

Usage:

setBeamMetaData [-e <energy>] [-t <particle>] [-l <folder>] <run>
Parameters
[in]<run>run number
[in]<energy>beam energy
[in]<particle>beam particle type
[out]<folder>DB folder the BeamMetaData information is stored in (default: "cd_calice_<beam location>/BeamMetaData")
