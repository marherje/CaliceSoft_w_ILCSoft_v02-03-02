Detailed Description

This tool dumps the HCAL temperatures from the database (channel-wise, module averages and calorimeter average). The values are averaged over a specified set of runs.

Usage:

extractHcalTemperatures <runs>
extractHcalTemperatures <runs> <outfile>
extractHcalTemperatures <runs> <infile> <outfile>
Parameters
[in]<runs>.txt file containing a list of runs for which the tempaeratures are extracted
[out]<outfile>(optional) if specified, the output is written to this .txt file instead of dumped to std::out
[in]<infile>(optional) if specified, the temperature averages are added to an existing calibration file
