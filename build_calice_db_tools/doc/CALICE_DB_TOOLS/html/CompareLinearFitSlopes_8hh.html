class  CompareLinearFitSlopes
 

Detailed Description

This tool compares two sets of CALICE::LinearFitSlopes.

Command line parameters:

–label1 -> define label for VAL1
–label2 -> define label for VAL2
–val1 -> define VAL1 input
–val2 -> define VAL2 input

–txt <inputfile.txt> -> define .txt file as source (format: module/chip/channel/value/error...)
–slcio <inputfile.slcio> <collection> -> define collection from .slcio file as source
–db <DBfolder> <time> -> define database folder at time <time> as source, using no tag
–db_tag <DBfolder> <time> <tag> -> define database folder at time <time> and tag <tag> as source
