Detailed Description

This tool writes a collection from an .slcio file to a database folder.

Usage:

relocateCollection [-r <infolder>] [-a <time>] [-l <outfolder>] [-f <from>] [-u <until>] [-k]
Parameters
[in]<infolder>database folder the collection is read from
[in]<time>read input database folder for this run or time
[out]<outfolder>database folder the collection is written to
[out]<from>DB entry will be valid from this point in time
[out]<until>DB entry will be valid until this point in time
-kswitch: keep the validity range of the input folder/collection

Any time string has to have the format YYYY-MM-DD-hh-mm-ss.sss
