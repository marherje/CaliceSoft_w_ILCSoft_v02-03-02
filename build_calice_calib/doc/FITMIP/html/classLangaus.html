double Langaus::findPeak ( const double *  par)

Find peak position for given parameters.

Precision of finding the peak position is given by number of points in Fast Fourier transformation (FFT) and by ratio of width of "binning" to range of fit for direct numerical computation. The number of points for FFT can be adjusted by setFFTCache and ratio by setNumRatio.

Parameters
par- pointer to array of three parameters: par[0]=Landau mean, par[1]=Landau sigma, par[2]=Gaus sigma
