Calculate error of peak position $A(x|\rho)$, where $\rho$ are parameters of function. 

The error is calculated with the formula $ \sigma^2_{peak}=\sum_{i,j} \frac{dA(x=peak|\rho)}{d\rho_i} \mbox{cor}(\rho_i,\rho_j)\cdot {\sigma_{\rho}}_i\cdot{\sigma_{\rho}}_j \frac{dA(x=peak|\rho)}{d\rho_j}$.
