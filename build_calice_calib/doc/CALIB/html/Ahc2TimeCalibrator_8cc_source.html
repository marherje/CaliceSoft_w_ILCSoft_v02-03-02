599 /*
600 Performs a linear fit to fit TDC values to the reference time for each
601 moduleNr, chipNr, channel, memoryCell, BxID
602 The slope is expected to stay constant on the chip, only being different for even and odd BxIDs.
603 The fit is done in two stages:
604 1. The first memoryCells of all channels of a chip are fitted with a robust fit in order
605 to not be influenced by outliers. From these fits the median slope is taken as the slope
606 for this chip.
607 The problem is, that the ROOT feature FixParameter() does not work with the reobust fit
608 for the current ROOT versions.
609 2. All hits that are further away than 10ns from the fit are rejected. Afterwards the
610 slope of the fit is fixed with the value calculated before and than all channels and
611 memorycells are again fitted (not robust) and the offset values of these fits are taken
612 */
