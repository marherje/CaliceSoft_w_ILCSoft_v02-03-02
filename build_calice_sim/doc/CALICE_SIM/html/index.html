CALICE SIM Documentation (v01-00)

This package contains classes to transform the simulated data of calice. into hits which can be handled by the classes in the calice reco package.. This comprises cell ganging (for the analogue hcal and the tail catcher) and the digitization step for the various sub detectors. Dependencies on other packages:

Note that this version contains the full cmake support.
