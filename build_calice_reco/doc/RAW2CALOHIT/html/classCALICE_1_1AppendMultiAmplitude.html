Detailed Description

Processor to add PAR_MULTI_AMPL (amplitude of the Multiplicity Counter) to the event.

This processor adds an event parameter containing the amplitude of the analog multiplicity counter readout. The mapping of the multiplicity counter connection is read from the LCIO stream.
