Detailed Description

Check for specific trigger configurations.

The class checks whether the trigger is configured for:

  • beam trigger
  • cosmics trigger
  • pedestal i.e. random trigger

The class depends on the trigger assignement and configuration conditions data. The configuration conditions data results from the raw-to-LCIO conversion.
