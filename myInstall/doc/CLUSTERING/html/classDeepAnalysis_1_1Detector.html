Constructor & Destructor Documentation

DeepAnalysis::Detector::Detector ( )
inline

Constructor.

The parameters are set to as default for HCAL (it is not valid for ECAL):

  • sampling = 30.0;
  • cell_size = 30.0;
  • ext_cell_size = 30.0;
  • neut_thresh = 0.6;
  • normal_thresh = 0.5;
  • elips_trans_to_catch= 1.5;
  • elips_forw_to_catch = 1.5;
  • elips_back_to_catch = 1.0;
  • ecc_trk_hadr_sel = 0.3;
  • a_delta = 0.01;
  • sph_cut = 0.05; // about 3 degree
  • cut_3d_em = 31.0;
  • cut_3d_had = 81.0;
  • join_factor = 1.0;
  • zoom_dist = 200.0.
