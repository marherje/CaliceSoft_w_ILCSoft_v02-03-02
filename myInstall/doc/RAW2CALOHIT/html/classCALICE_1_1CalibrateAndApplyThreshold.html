Detailed Description

Marlin Processor to go from RawCalorimeterHit to calibrated CalorimeterHits.

The hit position is calculated using information stored in a the cellID1. Moreover, the signal is calibrated if a proper calibration object exists. should run on Data as well as on MC after digisim.
