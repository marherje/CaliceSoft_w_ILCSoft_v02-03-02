12  /*
13  * Version 1.1: 6/2/2007
14  * DESY output set as TDC output (DC1 and DC2 swapped), CERN setup implemented.
15  *
16  * Version 1.0: 6/7/2006.
17  * First release of the processor, DESY setup implemented.
18  *
19  * Description of the processor:
20  * This processor performs the digitization of the MC hit produced in the
21  * Drift Chambers. The input collections are the four collection of hits in
22  * the DCs, the output collection is a collection such as the one produced
23  * by the lcio converter for the proper TDC. The processor smear the DC hits
24  * according to a given sigma.
25  *
26  * The options avilable are (updated to v1.1):
27  * DC1_MC_CollectionName Name of input DC1 Collection from MC
28  * DC2_MC_CollectionName Name of input DC2 Collection from MC
29  * DC3_MC_CollectionName Name of input DC3 Collection from MC
30  * DC4_MC_CollectionName Name of input DC4 Collection from MC
31  * TBTrackHitCollectionName Name of output TDC collection
32  * TBTrackTRUEHitCollectionName not used at the moment
33  * Cut_De_Dx Value above which hits in DC are used to calculate impact point
34  * Misallignment for misallignment studies only, to be set to 0
35  * Sigma Chamber resolution
36  * DriftVelocity Gas drift velocity (or propagation time for CERN DC)
37  * PrototypeModel Name of Prototype used (TBDesy0506,TBCern0806,..)
38  * Debug Set to 1 for debug, all cout enabled.
39  *
40  * DC collection from simulation in DESY:
41  * TBdch02_dchSD1 z = -1880
42  * TBdch02_dchSD2 z = -880
43  * TBdch02_dchSD3 z = -2780
44  * TBdch02_dchSD4 z = -3780
45  *
46  * Wire position in DESY (in TDC channel order: DC2,DC1,DC3,DC4)
47  *
48  * X1 and X3 is LEFT
49  * X2 and X4 is RIGHT
50  * Y1 and Y3 is DOWN
51  * Y2 and Y4 is UP
52  *
53  * Wire position in CERN
54  *
55  * 0, 4 and 2* is DOWN
56  * 1, 5 and 3* is UP
57  * 2, 0* and 5* is LEFT
58  * 3, 1* and 6* is RIGHT
59  * the * stand for the second TDC
60  *
61  * Autor: Michele Faucci Giannelli, Royal Holloway Univerisity of London
62  * email: giannell@pp.rhul.ac.uk or faucci@cern.ch
63  */
