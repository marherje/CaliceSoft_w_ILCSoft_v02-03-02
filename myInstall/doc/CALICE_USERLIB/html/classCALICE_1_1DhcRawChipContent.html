Detailed Description

Simple class which stores the dhcal chip content Note that the dchal as it is now delivers the same time stamp for all 64 channels served by a chip.

That's why the LCIO RawCalorimeterHit class is not convenient. Since it would lead to an increas of the data volume by a factor 64.
