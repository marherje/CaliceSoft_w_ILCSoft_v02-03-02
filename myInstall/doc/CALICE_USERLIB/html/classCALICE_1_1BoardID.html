Detailed Description

Create a packed board id of crate, slot, component numbers and the label.

Class is cheap and can be used to generate a packed board id: unsigned int packed_id = BoardID(short crateID, short slotID, short boardComponentNumber, short boardLabel).id(); or unsigned int packed_id = BoardID(short crateID, short slotID, short boardComponentNumber, short boardLabel);
