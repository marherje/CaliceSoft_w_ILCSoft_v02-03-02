Detailed Description

Class to store the DAQ address of a singel channel.

This class uses the mechanism of the BitField64 to encode and decode the bit positions using a string value. User defined encoding strings are possible. These have to define following fields:

  • crate The crate
  • slot Slot in the crate
  • fe Frontend on the card in slot
  • chip Chip on the frontend
  • channel Channel on the chip
