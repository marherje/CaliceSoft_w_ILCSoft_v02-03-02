Detailed Description

template<class T>
class CALICE::MappedContainer< T >

Container for fast access by index.

This class stores arbitrary data types and offers transparent access by different coordinate systems.

template<class T>
DecoderSet* CALICE::MappedContainer< T >::getDecoder ( )
inline

return a pointer to the DecoderSet handling the decoding of IDs with encoding strings

This function returns a pointer to the DecoderSet used by the Mapper. This allows to change the used encoding string. Be aware that the DecoderSet might be the same for many Objects using the same Mapper. Therefore, you should always enfore your encoding before using the MappedContainer.
