Detailed Description

Stores the configuration data of the front-end of one CERC board.

Based on AdcBlock. This is just a copy of the Calice DAQ class CrcFeConfigurationData. Since I don't have a profound understanding of the meaning of its members the documentation is mostly missing or at least very poor.
