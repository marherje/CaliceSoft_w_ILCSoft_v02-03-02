Detailed Description

Stores the general readout configuration.

e.g. to decide whether a softtrigger was enabled In addition information about general hardware configuration can be retrieved. For a give crate the enabled slots can be obtained by the getSlotEnable method For this slot in turn the enabled Fe can be found by withe getFeSlotEnable(int) method e.g. fe = getFeSlotEnable((slot-2)/4) byte = (slot-2)%4 => bits_enabled = (ife>>(8*byte))&0xff; The eight bits of enabled tells you which fe's were enabled in the given slot The eight LSB returned by the getFeSoftTrigger method tells you whether softtriggers were enabled for a given set of fe's.
