Detailed Description

Class to store float values for Saturation Correction Curves plus integer cell index in LCIO.

This class provides an interface to store four numbers inside an LCIO object:

  • an integer identifier, e.g. the index of a calorimeter cell
  • a 1st float value, [0], e.g. a maximum number of effective SiPM pixels
  • a 2nd float value, [1], e.g. a fraction of 2-exp function
  • a 3rd float value, [2], e.g. an epsilon of 2-exp function The 2-exp function is used: N = [0]*([1]*(1-exp(-[2]*x/[0]))+(1-[1])*(1-exp(-(1-[2]*[1])*x/([0]*(1-[1]))))) where x is the unsaturated amplitude, N is the saturated amplitude
