\subsection{Detailed Description}
Abstract base class of callback classes for the Root\-Tree\-Write processor Abstract interface for a Root\-Tree\-Writer-\/engine. Implement an descendant of this class to add variables to the output tree of the Root\-Tree\-Writer\-Processor. After that add the new {\itshape engine} to the list of writer engines in the \doxyref{Root\-Tree\-Writer}{p.}{classmarlin_1_1RootTreeWriter} processor
