\section*{Root\-Tree\-Writer}

This Package provides an Marlin processor to write a R\-O\-O\-T Tree with event variables. It provides a modular approach to extend the functionality. See the documentation of \doxyref{Root\-Tree\-Writer (Extending the Root\-Tree\-Writer)}{p.}{classmarlin_1_1RootTreeWriter} and \doxyref{Root\-Write\-Engine }{p.}{classmarlin_1_1RootWriteEngine} for details.
