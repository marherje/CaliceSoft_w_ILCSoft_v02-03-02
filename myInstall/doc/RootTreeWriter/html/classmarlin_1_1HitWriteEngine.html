void marlin::HitWriteEngine::fillVariables ( EVENT::LCEvent *  evt)
virtual

Implement this to extract the variables which you want to fill into the ROOT tree from the event. Write the value of the variables you want to add to the tree to the member variables, which you registered to the ROOT tree. The RootTreeWriter will call TTree::Fill() for you.

Parameters
evtthe current event

void marlin::HitWriteEngine::registerBranches ( TTree *  )
virtual 
Implement to register local variables to the output tree
\param pointer to the \a ROOT tree, which the RootTreeWriter processor fills at the
end of each event.
Usually the implementation looks like@verbatim

/// hostTree->Branch("variable",&_variable" ,"variable/F" ); /// . But you can add any type of branch you like.
