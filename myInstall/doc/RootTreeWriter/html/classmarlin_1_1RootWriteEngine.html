Detailed Description

Abstract base class of callback classes for the RootTreeWrite processor Abstract interface for a RootTreeWriter-engine. Implement an descendant of this class to add variables to the output tree of the RootTreeWriterProcessor. After that add the new engine to the list of writer engines in the RootTreeWriter processor

virtual void marlin::RootWriteEngine::fillVariables ( EVENT::LCEvent *  evt)

Implement this to extract the variables which you want to fill into the ROOT tree from the event. Write the value of the variables you want to add to the tree to the member variables, which you registered to the ROOT tree. The RootTreeWriter will call TTree::Fill() for you.
