\section{Introduction}\label{index_whatsthis_sec}
This is a set of tools for extracting information from the C\-A\-L\-I\-C\-E conditions database. A description of the individual programs can be found \doxyref{here}{p.}{group__tools_exec_doc}.

All tools use a default init string for read access to the database. If you need a special init string, you can set it by defining the environment variable C\-A\-L\-I\-C\-E\-\_\-\-D\-B\-\_\-\-I\-N\-I\-T (export C\-A\-L\-I\-C\-E\-\_\-\-D\-B\-\_\-\-I\-N\-I\-T=\char`\"{}...\char`\"{}).
