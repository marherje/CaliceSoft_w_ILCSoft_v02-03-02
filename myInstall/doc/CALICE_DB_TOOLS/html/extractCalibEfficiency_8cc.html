Detailed Description

This tool calculates the efficiency of calibration sets taken in a certain run range and creates TGraphErrors from these values vs. runTime. The total efficiency is calculated as well. Dead channels are subtracted from channel total of 7608 (AHCAL).

Usage:

extractCalibEfficiency <dbfolder-calib> <tag-calib> <first> <last> <minium efficiency in %> [<dbfolder-exclude> <tag-exclude>]
Parameters
[in]<dbfolder-calib>DB folder containing results from the single calib measurements (e.g. "/cd_calice/Hcal/Calib_Measurements/gain_single")
[in]<tag-calib>tag for DB folder containing results from the single calib measurements
[in]<first>first run of run range
[in]<last>last run of run range
[in]<minimumefficiency in %> measurements with efficiencies below this cut are negelected
[in]<dbfolder-exclude>DBfolder containing list of channels to exclude from efficiency calculation (e.g. dead channels)
[in]<tag-exclude>tag for DBfolder containing list of channels to exclude from efficiency calculation (e.g. dead channels)
