Detailed Description

This tool writes the content of a database folder for a given tag and point in time to an .slcio file.

Usage:

copy_dbfolder2simplefile [-t <tagname>] [-a <time>] <foldername> <filename>
Parameters
[in]<foldername>source database folder
[out]<filename>output file (.slcio)
[in]<tagname>(optional) choose specific folder tag (use HEAD if left empty)
[in]<time>point in time at which the DB information is extracted
