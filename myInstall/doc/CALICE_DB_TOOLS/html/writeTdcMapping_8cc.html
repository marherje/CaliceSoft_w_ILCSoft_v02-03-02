Detailed Description

This tool creates a TdcMapping object and writes it to an LCCD conditions database. Input values have to be given as command line arguments and default to zero.

Usage:

writeTdcMapping [-l <folder>] [-f <from>] [-u <until>] [-d <descr>] [-r <file>] [-w]
Parameters
[out]<folder>database folder the collection is written to
[out]<from>DB entry will be valid from this run or point in time
[out]<until>DB entry will be valid until this run or point in time
[out]<descr>description of DB entry
[in]<file>file to read input data from
<-w>switch: nothing is written without -w (test run)

Any time string has to have the format YYYY-MM-DD-hh-mm-ss.sss
