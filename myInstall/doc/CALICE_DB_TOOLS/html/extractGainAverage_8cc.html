Detailed Description

This tool averages calibration parameters (e.g. gain or mip) for each channel over a list of runs and dumps the results to std::out.

Usage:

extractGainAverage <dbfolder> <temperatureCalib> <first> <last> [<exclude>]
Parameters
[in]<dbfolder>DB folder containing results from the single calibration measurements (e.g. "/cd_calice/Hcal/Calib_Measurements/gain_single")
[in]<tempCalib>DB folder containing calibration for temperature sensors (e.g. "/cd_calice/Hcal/tempSensors")
[in]<first>first run of run range that is used for calculating the average
[in]<last>last run of run range that is used for calculating the average
[in]<exclude>(optional) .txt file with list of runs that are excluded from average
