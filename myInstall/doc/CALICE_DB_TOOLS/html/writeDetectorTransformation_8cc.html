Detailed Description

This tool creates a CALICE::DetectorTransformation object and writes it to an LCCD conditions database. Input values have to be given as command line arguments and default to zero.

Usage:

writeDetectorTransformation [-l <folder>] [-f <from>] [-u <until>] [-d <descr>] [-w] [-a <angle>] [-x <x>] [-y <y>] [-z <z>] [-c]
Parameters
[out]<folder>database folder the collection is written to
[out]<from>DB entry will be valid from this run or point in time
[out]<until>DB entry will be valid until this run or point in time
[out]<descr>description of DB entry
<-w>switch: nothing is written without -w (test run)
[in]<angle>rotation angle
[in]<x>x-position
[in]<y>y-position
[in]<z>z-position
<-c>switch: units are given in cm instead of mm

Any time string has to have the format YYYY-MM-DD-hh-mm-ss.sss
