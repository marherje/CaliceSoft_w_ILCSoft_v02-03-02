Detailed Description

This is a Marlin processor for displaying events with CED.

If you want to display events with CED, you need to start the display (glced) before running Marlin.

Mouse commands that can be used within CED:

  • left + move -> rotate view
  • right + move down/up -> zoom view in/out
  • center + move -> shift view
  • wheel -> shift view along z-axis

Keyboard commands that can be used within CED:

  • b -> toggle background color
  • r -> got to default angular view
  • s -> got to default side view
  • f -> got to default front view
  • v -> switch on/off fisheye view
  • 0 ... 9 -> switch on/off CED display layer 0 ... 9
  • shift + 0 ... 9 -> switch on/off display layer 10 ... 19
  • t -> switch on/off CED display layer 20
  • y -> switch on/off CED display layer 21
  • u -> switch on/off CED display layer 22
  • i -> switch on/off CED display layer 23

Assignment of CED display layers:

  • 0:
  • 1: HBU / EBU hits ( hit energy < 0.5 mip )
  • 2: HBU / EBU hits ( 0.5 mip <= hit energy < 1.65 mip )
  • 3: HBU / EBU hits ( 1.65 mip <= hit energy < 2.9 mip )
  • 4: HBU / EBU hits ( 2.9 mip <= hit energy < 5.4 mip )
  • 5: HBU / EBU hits ( hit energy >= 5.4 mip )
