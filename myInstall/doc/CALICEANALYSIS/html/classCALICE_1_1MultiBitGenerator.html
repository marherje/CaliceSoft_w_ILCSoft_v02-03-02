Detailed Description

Processor which generates a trigger bit for particle multiplicity (PAR_VETO_BIT) based on the multiplicity counter amplitude given by PAR_MULTI_AMPL. The threshold values can be extracted from the database based on a chosen purity for the single-particle-sample (default) or be defined explicitly in the steering file (set "useFixedThreshold" to "true").
