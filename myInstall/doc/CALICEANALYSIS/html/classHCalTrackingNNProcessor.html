Detailed Description

This marlin processor tries to find tracks in the calorimeter. Currently optimised for the AHCal of CALICE.

The algorithm works the following way:

  1. Check all hits whether they are isolated (e.g. have no neighbour hits in the same layer)
  2. Take an isolated hit and advance to the next layer. Check, if there is an isolated hit at the same position. If there is: join them together to a track.
  3. For analysis, the energy distribution for all found tracks is saved. (both total, and per tile)
