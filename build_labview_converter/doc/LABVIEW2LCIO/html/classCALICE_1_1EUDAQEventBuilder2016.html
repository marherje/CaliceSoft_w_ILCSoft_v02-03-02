Detailed Description

Processor for reading the AHCAL EUDAQ SLCIO files.

It processes the input data to create events with LCIO collections of the AHCAL EUDAQ SLCIO output (sorted by readout cycles). Based on F. Gaede's StdHepReader to be found in the marlin package.
