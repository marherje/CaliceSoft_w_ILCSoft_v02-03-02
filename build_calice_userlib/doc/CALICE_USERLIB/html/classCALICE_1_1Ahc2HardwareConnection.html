Detailed Description

Class to store int values for HardwareConnection plus integer cell index in LCIO.

This class provides an interface to store four numbers inside an LCIO object:

  • an integer identifier, e.g. the index of a calorimeter cell
  • a 1st int value, ChipID
  • a 2nd int value, ModuleNumber
  • a 3rd int value, ChipNumber

This class is used in Ahc2CalibrateProcessor to get ModuleNumber and ChipNumber from Hardware information
