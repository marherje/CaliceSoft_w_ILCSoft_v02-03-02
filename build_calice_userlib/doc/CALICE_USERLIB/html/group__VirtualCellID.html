virtual int CALICE::Mapper::getTrueCellID ( const int  virtualCellIndex) const
pure virtual

return identifying index of cell

Cells of different size span different areas in Mokka CellID space. This functions returns for all coordinates of the cell the Mokka CellID which is used to uniquely identify it.
