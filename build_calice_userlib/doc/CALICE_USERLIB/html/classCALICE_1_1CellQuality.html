Detailed Description

LCIO contions data class to describe the cell quality.

The cell can be dead or noisy (or both, for instance if a cell is noisy and then dies during a run period).

Note: Internally the status is treated as a bit pattern, which allows setting several flags simulaneously and can easily be extended. Thanks to the user interface the user never has to fiddle around with the bits. The definition is protected inside the class and queried with isDead() or isNoisy().
